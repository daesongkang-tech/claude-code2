# TikTokリール動画コンテンツ自動生成

集客目的のTikTokリール動画（15〜60秒）のコンテンツ一式を生成する。引数（`$ARGUMENTS`）にテーマ・サービス名・ターゲット層が渡された場合はそれを優先する。指定がなければコンテキストから最適なテーマを選ぶ。

## 手順

1. **ターゲット・テーマの確定**
   - `$ARGUMENTS` が渡された場合はそれをテーマとして使用する
   - 渡されていない場合は `company/` 配下のコンテキストを参照し、現在の注力サービス（ビズフリ・ビズプロ・AX等）から最適なテーマを選定する
   - ターゲット層（フリーランス志望、転職検討中、DX推進担当者など）を明確にする

2. **フックの作成（冒頭0〜3秒）**
   - 視聴者が止まるキャッチーな一文を3パターン作成する
   - 数字・疑問・共感のいずれかの型を使う（例：「月収50万のフリーランスがやっている3つのこと」）

3. **台本の作成（本編）**
   - 構成：フック → 問題提起 → 解決策の提示 → CTA（行動喚起）
   - 尺：15秒・30秒・60秒の3バリエーションで作成する
   - テロップ用テキストと話者のセリフを分けて記載する

4. **キャプション・ハッシュタグの作成**
   - キャプション：1〜2文（簡潔でCTAを含む）
   - ハッシュタグ：日本語・英語を混在させて15〜20個（業界タグ・バズタグ・ブランドタグのバランスを考慮）

5. **撮影・編集メモの作成**
   - 推奨カット割り・BGMの雰囲気・テロップのフォント感・カラートーンを簡潔に記載する
   - 撮影時の注意点（縦型9:16、明るさ、話すスピード等）を箇条書きにする

6. **アウトプットの出力**
   - 以下の形式で出力する：

```
## TikTokリール動画コンテンツ

**テーマ**: [テーマ名]
**ターゲット**: [ターゲット層]
**推奨尺**: [秒数]

---

### フック案（冒頭3秒）
1. [フック案1]
2. [フック案2]
3. [フック案3]

---

### 台本（30秒版）
[セリフ＋テロップ構成]

### 台本（60秒版）
[セリフ＋テロップ構成]

---

### キャプション
[キャプションテキスト]

### ハッシュタグ
[ハッシュタグ一覧]

---

### 撮影・編集メモ
[撮影メモ箇条書き]
```
